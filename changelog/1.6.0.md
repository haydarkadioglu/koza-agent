  - **Focus Modes** (Developer, Researcher, Designer, Freestyle) which adjust the UI accent colors.
  - **Quick Action Buttons** on code blocks (Run in Terminal, Apply to File).

- Removed 'DEV' channel indicator from the titlebar for a cleaner look.
- Bumped version across all workspace packages to 1.6.0.
