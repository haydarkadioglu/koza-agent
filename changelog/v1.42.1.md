# v1.42.1 — Voice, Typer CLI, and Better Build Output

## New Features

- Added a dedicated Voice / STT / TTS setup flow in `koza setup` and `koza voice setup`.
- Speech-to-text and text-to-speech providers can now be configured separately.
- Added STT provider support for Local Whisper, OpenAI, Gemini, and Deepgram.
- Added TTS provider support for System pyttsx3, Kokoro ONNX, OpenAI, Gemini, and ElevenLabs.
- Added `DEEPGRAM_API_KEY` and `ELEVENLABS_API_KEY` environment variable support.
- Added a Typer-based CLI command tree with typed options, improved help output, shell completion support, and backward-compatible command aliases.

## Improvements

- Added Skip / keep current options throughout the voice setup flow so accidental menu entries do not overwrite existing settings.
- Preserved backward compatibility with legacy voice config keys such as `voice.stt_model`, `voice.tts_voice`, `voice.use_kokoro`, and `voice.language`.
- Voice runtime now dispatches to the selected provider and reports missing API keys or dependencies with short actionable errors instead of silently falling back.
- Improved build intent routing so requests like "website yap", "React portfolio oluştur", and "landing tasarla" are treated as coding/build tasks instead of web research.
- Added a minimum website/app quality floor to prevent one-title placeholders and require real sections, responsive styling, meaningful content, and verification.

## Files Modified

- `config.py` — Added nested voice provider defaults and Deepgram / ElevenLabs provider config.
- `cli/setup.py` — Routes setup voice configuration through the shared Voice / STT / TTS helper.
- `cli/voice_cmd.py` — Added shared voice setup menus and provider-specific STT/TTS choices.
- `skills/voice.py` — Added STT/TTS provider dispatch for Local Whisper, OpenAI, Gemini, Deepgram, Kokoro, System TTS, and ElevenLabs.
- `koza_run.py` / `cli/typer_app.py` — Added Typer CLI entrypoint and backward-compatible command wrappers.
- `prompt.py` / prompt sections — Added stronger build-output quality rules.
