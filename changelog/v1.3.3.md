# v1.3.3 — 2026-05-21

## ⌨️ CLI — Interrupt + Type While Processing

### Write while agent is running
- With `prompt_toolkit`: input field is now always active even while the agent is processing
- Pressing **Enter** on an empty input interrupts; typing a message and pressing **Enter** interrupts *and* immediately queues the new request
- Prompt label updated: `⏎  interrupt  │  type to queue next ›`

### Fallback mode (no prompt_toolkit) improved
- Agent now runs in a background thread in fallback mode too — terminal is no longer blocked
- **Ctrl+C** interrupts the current request; next prompt appears immediately

## 🛑 Agent Loop — PERMANENT FAILURE guard

### Core loop fix (`core.py`)
- When any tool result contains `PERMANENT FAILURE`, the agent loop breaks immediately
- Model is told once to report the error to the user — no retry with alternative prompts
- Fix applies to both `stream_chat` (streaming) and `chat` (non-streaming) paths
- Eliminates the "same error, different prompt" retry pattern for hard failures like Playwright timeouts
