# v1.4.3

## 🛑 Interrupt & Stability — Real Cancel, Tool Calls in Stream, UI Fixes

Interrupt artık kozmetik değil, gerçek. Tüm provider'larda HTTP bağlantısı kesilir, token tüketimi durur. Ayrıca stream modunda kaybolan tool call'lar düzeltildi ve UI input corruption sorunları çözüldü.

### New Features

- **Real Cancel Support (All Providers)** — `cancel_event` parametresi base provider interface'e eklendi; tüm 8 provider'da `cancel_event.is_set()` kontrolü + `resp.close()` ile HTTP bağlantısı anında kesilir
- **Immediate "Interrupting…" Feedback** — Interrupt tetiklendiğinde status bar anında "⏸ Interrupting…" gösterir, agent'ın durmasını beklemez
- **5-Second Force-Kill Watchdog** — Interrupt 5 saniye içinde tamamlanmazsa tüm state force-clear edilir ve "⚠ Force stopped" mesajı gösterilir
- **Throttled UI Invalidation** — Streaming sırasında `invalidate()` çağrıları 100ms'ye throttle edildi; 50+ token/sn hızında redraw sayısı ~10/sn'ye düşürüldü

### Bug Fixes

- **Fixed: Tool calls lost in stream mode (OpenAI, Anthropic, Ollama)** — Tool call delta'ları artık stream sırasında buffer'lanıp `__tool_chunk__` dict olarak yield ediliyor
- **Fixed: DeepSeek buffers entire response** — Text token'ları artık geldiği anda yield ediliyor; DSML cleanup post-stream'e taşındı
- **Fixed: Input text disappears while agent is streaming** — `dont_extend_height=True` ile TextArea yükseklik recalculation'ı engellendi
- **Fixed: invalidate() from background threads corrupts input** — Tüm `invalidate()` çağrıları `call_soon_threadsafe` üzerinden yönlendirildi
- **Fixed: Permission UI corrupts prompt_toolkit display** — `run_in_terminal()` wrapper ile TUI geçici olarak suspend ediliyor
- **Fixed: Zombie thread on submit-while-busy** — `_stream_lock` ile concurrent `stream_chat` engellendi; 5s join timeout + force-clear fallback eklendi
- **Fixed: Generator state leak on interrupt** — `CodingSession` aktif generator'ı `gen.close()` ile kapatıyor, tüm sub-agent'larda `_busy = False` zorlanıyor

### Provider Changes

| Provider | Before | After |
|----------|--------|-------|
| DeepSeek | Full buffer, no cancel | Incremental yield + cancel + DSML post-process |
| OpenAI | No tool calls in stream, no cancel | Tool call buffering + cancel |
| Anthropic | No tool calls in stream, no cancel | Event-based tool_use capture + cancel |
| Ollama | No tool calls in stream, no cancel | JSON line tool_call buffering + cancel |
| Gemini (api_key) | No cancel | Cancel check in stream loop |
| Groq | No cancel | Cancel check + resp.close() |
| OpenRouter | No cancel | Cancel check + resp.close() |
