# v1.3.2 — 2026-05-21

## 🖼️ Media Generation — Setup Overhaul

### Advanced Media Setup
- `koza setup` → Media section rebuilt with per-section **Skip** option
- Gemini Browser Session flow integrated: session status checked live, re-login option offered if session exists
- OpenAI (DALL-E 3) and Gemini API (Imagen 3) options retained

### `koza generate` command removed
- Image/video generation is now requested conversationally through the agent (e.g. "generate an image of…")
- Keeps the CLI surface clean

### Voice Mode — disable fixed
- `koza setup` → Voice Mode → **Disable** now correctly persists `enabled: false` to config

### Gemini image/video error messages improved
- Timed-out or failed Playwright operations return `❌ PERMANENT FAILURE — do not retry.` prefix
- Agent loop now hard-stops on `PERMANENT FAILURE` instead of retrying with a different prompt
