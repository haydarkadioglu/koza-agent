# v1.4.4

## 📝 Prompt Externalization — Harici Prompt Yönetimi & Self-Sufficiency Analizi

Tüm hardcoded system prompt'lar Python kaynak dosyalarından çıkarılıp `prompts/` dizininde harici `.md` dosyalarına taşındı. Artık prompt'lar kod değiştirmeden düzenlenebilir, versiyonlanabilir ve bağımsız test edilebilir. Ayrıca agent'ın otonom çalışma kapasitesini artırmak için kapsamlı bir self-sufficiency analizi eklendi.

### New Features

- **PromptLoader Module** — Singleton pattern, mtime-based cache invalidation, thread-safe (threading.Lock) prompt yükleme sistemi
- **External Prompt Files** — 18 adet `.md` dosyası: core (1), sections (9), personas (4), channels (4), routing (1)
- **Hot-Reload Support** — Prompt dosyaları düzenlendiğinde restart gerekmez; mtime değişikliği otomatik algılanır
- **Dynamic Section Discovery** — `prompts/sections/` dizinine yeni `.md` dosyası eklemek yeterli, kod değişikliği gerekmez
- **Self-Sufficiency Analysis** — `docs/self-sufficiency-analysis.md` — eksik tool kapasiteleri, error recovery, context management, prompt optimization analizi

### Refactoring

- **prompt.py** — `CORE_PROMPT`, `PROMPT_SECTIONS`, `CHANNEL_PROMPTS` artık PromptLoader üzerinden yükleniyor; ~150 satır inline string kaldırıldı
- **router.py** — `_ROUTING_SYSTEM_PROMPT` artık `prompts/routing/classifier.md` dosyasından yükleniyor
- **skills/agents/personas.py** — 4 persona prompt'u (~200 satır) kaldırılıp loader'dan yükleniyor
