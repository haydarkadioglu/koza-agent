ชุดโครงการชุมชนที่สร้างขึ้นบน Koza

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ Koza ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-koza](https://github.com/awesome-koza/awesome-koza) และ [koza.cafe](https://koza.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
