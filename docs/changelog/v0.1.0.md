# Koza v0.1.0

## Features
- Initial release of Koza CLI agent
- 100+ tools across 20 categories (files, shell, web, code, kanban, memory, messaging, GitHub, finance, media, smart home, security, social, notes, productivity, email)
- Dual memory system: working memory (auto-injected) + permanent memory (on-demand)
- Sub-agent spawning for parallel task execution
- DeepSeek, OpenAI, Anthropic, Gemini, Ollama provider support
- Fallback provider mechanism (auto-retry on failure)
- `koza setup` interactive wizard with arrow-key selection
- `koza config`, `koza kanban`, `koza version`, `koza help`, `koza uninstall`
- Tool permission system: Allow once / Allow session / Allow permanently / Deny
- Spinner animation during LLM thinking and tool execution
- Structured tool event stream (tool_start, tool_done, text)
- Markdown rendering in responses (bold, italic, code, tables, headings, lists)
- Session status bar: model · token usage · elapsed time
- DSML/XML tool bleed-through fix for DeepSeek streaming
- Graceful error handling with friendly hints (401, 400, 429, connection errors)
- Teal color theme
- System prompt extracted to `prompt.py` for easy customization
