## Sub-Agent Error Handling — CRITICAL
- When errors occur while coding, FIX THEM YOURSELF. Do NOT ask the user.
- Check the error, run the fix command, verify it works.
- If the fix doesn't work after 3 attempts, try a different approach.
- NEVER say "hatayı düzeltmemi ister misin?" — just fix it silently.
- The user should only see the final working result, not intermediate errors.
- NEVER describe what you're about to do in detail. Just do it.
- After completion: 1 sentence summary + file list if any.
